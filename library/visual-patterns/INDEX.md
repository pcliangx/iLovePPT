# Visual Patterns Index

> 给人和 LLM grep 的速查索引。每 ingest 一个新 pattern, 在此加一行。
> RAG 检索请用 `library/search.sh`。

## process(流程)

- `timeline-h-5` · 横向 5 阶段时间线(项目 / roadmap / milestone)
- `timeline-v-4` · 纵向 4 阶段时间线(Q1-Q4 / 上下排版)
- `process-arrow-5` · 5 步箭头 chevron 流程链
- `funnel-4-tier` · 4 层漏斗(销售 / 招聘 / 转化漏斗)

## cycle(循环)

- `pdca-4-cycle` · PDCA 4 节点环形循环(Plan/Do/Check/Act)

## comparison(对比)

- `swot-quadrant` · SWOT 4 象限战略分析
- `2x2-priority-matrix` · 2x2 优先级矩阵(Eisenhower)
- `cards-flag-4` · 4 个 flag 卡片(数字 / icon + 标题 + 正文)
- `comparison-tier-3` · 3 级套餐对比(Basic/Pro/Enterprise + 推荐项)
- `bullet-5-numbered` · 5 项编号列表(01-05 编号)

## hierarchy(层级)

- `pyramid-maslow-5` · Maslow 5 层金字塔
- `pyramid-3-layer` · 3 层金字塔(基础 / 核心 / 顶尖)

## data(数据)

- `single-focus-kpi` · 单大数字 KPI(超大数字 + 一行说明)

## relationship(关系)

- `radial-5-spoke` · 5 辐射(中心 + 5 节点 hub-and-spoke)
- `venn-3-circle` · 3 圆相交维恩图(交集 / sweet spot)
